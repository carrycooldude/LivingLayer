# Visual Identity

LivingLayer should feel platform-level, GPU-native, Android-first, holographic, and premium.

## Reference Space

- Nothing OS precision and restraint.
- visionOS depth and material layering.
- PS5 UI glow, depth, and focus behavior.
- WinUI acrylic material language.
- Qualcomm demo aesthetics: performance-forward, dimensional, technical.

## What It Is

LivingLayer is a reactive Android graphics framework. It is not a single icon effect.

The identity should communicate:

- Hardware acceleration.
- Dimensional UI surfaces.
- Sensor-aware motion.
- Glass, glow, refraction, reflection, and bloom.
- A path from Compose UI to lower-level graphics backends.

## What To Avoid

- Cute sample-app animation language.
- Toy-like icon treatments.
- Overly soft pastel gradients.
- Single-effect branding.
- Decorative motion that does not reveal depth, state, or material.

## Color Direction

Use deep graphite and near-black bases with controlled luminous accents:

- Graphite: `#090B10`
- Carbon: `#11151C`
- Hologram cyan: `#58E6FF`
- Spectral blue: `#6C7CFF`
- Plasma violet: `#B86CFF`
- Precision white: `#F4F7FB`

The palette should feel calibrated and technical rather than candy-colored.

## Motion Direction

Motion should suggest real material response:

- Subtle gyroscope parallax.
- Focus bloom on interaction.
- Refraction shift under touch.
- Scanline or spectral shimmer only when it supports the material.
- Fast return-to-rest behavior.
