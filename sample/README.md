# LivingLayer Sample App

This Android app consumes the published Maven Local package:

```kotlin
implementation("io.livinglayer:livinglayer:1.0.0-alpha01")
```

Run:

```powershell
.\gradlew.bat publishToMavenLocal :sample:installDebug
```

Recommended first screens:

- LivingIcon grid with tilt response.
- Glass material card.
- Hologram shader demo.
- Ripple touch demo.
- Performance overlay.
