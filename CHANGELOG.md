# Changelog

## 1.0.0-alpha02

- Adds live accelerometer parallax to `LivingIcon` when `gyroscope = true`.
- Adds idle shimmer/glow motion so sample surfaces visibly move even when the phone is still.
- Updates the sample app dependency and package version.

## 1.0.0-alpha01

Initial LivingLayer package preview.

- V1 reactive layer engine, Compose API, effects, shaders, and motion engines.
- V2 shader graph model and builder API.
- V3 render backend abstraction with Compose backend and Vulkan extension placeholder.
- V4 launcher scene engine for adaptive Android surfaces.
- V5 adaptive interface generation engine with deterministic local policy.
- Maven publication metadata for local and remote package release.
